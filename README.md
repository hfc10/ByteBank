## 📌 Sobre o Projeto

O **ByteBank** é uma aplicação web que tem como objetivo auxiliar usuários no controle de suas finanças pessoais, permitindo o registro, organização e análise de receitas e despesas.

O sistema oferece uma visão clara da situação financeira através de dashboards, gráficos e histórico de transações, facilitando o planejamento financeiro.

---

## 🚀 Funcionalidades

- 🔐 Cadastro de usuários
- 🔑 Sistema de login  
- 💸 Registro de receitas e despesas  
- ✏️ Edição e exclusão de transações  
- 📊 Dashboard com resumo financeiro  
- 📈 Gráficos de gastos por categoria  
- 🧾 Histórico de transações    

---

## 🏗️ Arquitetura do Sistema

O projeto foi desenvolvido com:

- Arquitetura em **3 camadas**
- Backend estruturado no padrão **MVC**
- Comunicação via **API REST**
